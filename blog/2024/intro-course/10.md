## 累積和とは

累積和とは、前処理をしておくことで、計算量を少なく指定された区間の和を出す手法のことです。

## 例題

<https://atcoder.jp/contests/nikkei2019-final/tasks/nikkei2019_final_a>

この問題を累積和で解くと以下のようになります。

```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
    long long n, mx=0;
    cin >> n;
    vector<long long> a(n+1);
    for ( int i = 0; i < n; i++ ) {
        cin >> a.at(i+1);
    }
    for ( int i = 1; i < n+1; i++ ) {
        a.at(i) += a.at(i-1);
    }
    for ( int k = 1; k < n+1; k++ ) {
        for ( int i = k; i < n+1; i++ ) {
            mx=max(a.at(i)-a.at(i-k),mx);
        }
        cout << mx << endl;
    }
}
```

何をしているのかというと、

```cpp
    for ( int i = 1; i < n+1; i++ ) {
        a.at(i) += a.at(i-1);
    }
```

の部分でそこまでの和を計算しています。  
そして求めたい範囲の和を、初めから範囲の前までの和と、初めから範囲の最後までの和の差によって求めています。
